Front End Installation
1. Install Node JS (https://nodejs.org/en)
2. Install Angualar version 14.2.0(https://www.npmjs.com/package/@angular/cli/v/14.2.0)
3. Run ng server inside the project folder

Backend installation

1. install Node js
2. Install Npx Ts node(https://www.npmjs.com/package/ts-node)
3. Run npx npx ts-node src/server.ts


General Working Flow

1. Login as Manager(Username: manager1, password: manager1)
2. Register a new doctor
3. Then Logout
4. Login as Docotor using the credentials that are created while registering
5. Create some services for the doctor
6. Logout
7. Register a new patient
8. Login as Patient
9. Click Doctor or appointments
10. Choose a doctor
11. Select particular service, date and enter remarks  12. and click on book
12. Now logout, and login again as the doctor to see the bookings.
